# Portfolio

Personal Full Stack Developer Portfolio showcasing projects, skills, and experience in React, Next.js, Node.js, and DevOps.

## Getting started

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app. Edit `src/app/page.tsx` to change the home page.

Built with [Next.js](https://nextjs.org).
